# 线性DP

## 概述

| 模型                 | 特点                                                         | 时间复杂度 | 经典题目 |
| -------------------- | ------------------------------------------------------------ | ---------- | -------- |
| 数字三角形（路径DP） | 以图为背景，每一点的dp值与相邻点的dp值有关                   | `O(nm)`    |          |
| LIS（序列DP）        | 以数字序列为背景，每一位的dp值与某一方向上前面每一位的dp值有关 | `O(n^2)`   |          |
| LCS（序列DP）        | 以字母序列为背景，每一位的dp值与某一方向上前面每一位的dp值有关 | `O(n^2)`   |          |



## 技巧与经验

| 技巧     | 使用场景                                     | 经典题目 |
| -------- | -------------------------------------------- | -------- |
| 贪心探路 | 未定最优化问题可以使用贪心探路，如果失败换DP |          |
| 贪心优化 | 优化DP环节中的时间复杂度                     |          |
